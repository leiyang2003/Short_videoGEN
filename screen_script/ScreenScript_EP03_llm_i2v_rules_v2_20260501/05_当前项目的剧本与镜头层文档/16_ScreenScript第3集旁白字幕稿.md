# ScreenScript第3集旁白字幕稿

> 项目标题：四年前那个夜晚（下）

- SH01：字幕：四年前 · 两个月后 一间不到二十平的小公寓。墙皮有些剥落，窗帘是地摊上最便宜的那种。但收拾得很干净，桌上摆着一小盆绿萝。 沈念歌坐在床边，手里握着一根验
- SH02：手机响了。屏幕上：林雨薇。 她没有接。 手机继续响。第二条消息：不可读文字块 沈念歌盯着那条消息看了很久。 然后，她把手机翻过去，屏幕朝下扣在桌上。，一部手机首
- SH03：我不知道你是谁的。但你是我的。
- SH04：予予……妈妈给你取的名字，知予。知道的知，给予的予。妈妈知道的一切，都想给你。
- SH05：沈小姐！你产后还没恢复，不能出院——而且孩子的出生登记——父亲那一栏你空着，需要补充——
- SH06：父亲不详。
- SH07：我说了。不详。
- SH08：她抱着孩子，走进了小城的巷子里。 巷子很窄，两侧是老旧的墙壁。阳光从头顶的天空洒下来，照在她单薄的远去身影上。 她走得很慢，但每一步都很稳。
- SH09：【画面一】深夜，一间出租屋。沈念歌一手抱着哭闹的予予，一手在电脑上打字做兼职翻译。桌上是泡面和几罐奶粉。时钟指向凌晨三点。 【画面二】清晨，菜市场。沈念歌蹲在菜
- SH10：祝你生日快乐……祝你生日快乐…… / 不能碰，会烫。
- SH11：妈咪，那个房子好大！我们以后能住那么大的房子吗？
- SH12：我爸爸……在很远的地方。但我觉得他会来找我。
- SH13：字幕：一周前 予予趴在桌上画画。沈念歌坐在旁边看手机——滨海市儿童医院心内科的预约确认信息。，一部手机首帧可见，屏幕朝向持有者，屏幕内容不可见
- SH14：妈咪要带你回滨海市。
- SH15：滨海市？就是那个有大高楼的城市？
- SH16：嗯。那边有个很好的医生，能帮予予把心脏上的小蝴蝶修好。
- SH17：妈咪，滨海市……是不是我爸爸在的地方？
- SH18：……我不知道。
- SH19：我觉得是。我有预感。
- SH20：四年了。我以为这辈子都不会再回到那个城市。可是予予需要最好的医生。而最好的医生……在滨海市。
- SH21：她站起来，开始收拾行李。行李很少——两个箱子，一箱是予予的衣服和药，一箱是她仅有的几件衣服。 予予跑过来，从口袋里掏出一张画。
- SH22：妈咪你看！这是我们一家三口！中间这个黑色的是爸爸，因为我还没见过他！等见到了我就涂成肉色！
- SH23：好。等找到了，你涂成最漂亮的颜色。
