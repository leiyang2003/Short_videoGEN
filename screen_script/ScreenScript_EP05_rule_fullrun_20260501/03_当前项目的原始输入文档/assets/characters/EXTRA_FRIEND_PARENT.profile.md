# 闺蜜 角色档案

【基本身份背景】
角色叫闺蜜，三十多岁，主要活动在现代中国滨海市都市短剧，风衣妈妈旁边的幼儿园家长闺蜜，三十多岁女性，精致都市穿搭，旁侧听众气质，不能与风衣妈妈同脸。
