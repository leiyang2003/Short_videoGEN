# 部门经理 角色档案

【基本身份背景】
角色叫部门经理，三十多岁，主要活动在现代中国滨海市都市短剧，会议室临时部门经理，深色商务装，只作为汇报功能人物。
部门经理是短剧中的关键角色，视觉锚点为：会议室临时部门经理，深色商务装，只作为汇报功能人物。

【容貌】
形象定位标签：会议室临时部门经理，深色商务装，只作为汇报功能人物。
年龄观感：三十多岁
脸型骨相：脸型骨相必须与同项目其他角色区分，避免复用默认美型脸
五官特征：会议室临时部门经理，深色商务装，只作为汇报功能人物
发型：发型必须来自角色身份和时代，不与同项目同性别角色重复
体态/身高感：体态和站姿体现职业、阶层和心理状态
服饰主锚点：服装主锚点符合角色身份、职业和故事环境
服饰颜色/材质：服饰颜色和材质应稳定，并与其他角色形成区分
职业/阶层细节：会议室临时部门经理，深色商务装，只作为汇报功能人物
表情默认值：表情符合人格锚点：场景功能、临时反应
与其他角色的区别：必须和同项目其他角色在脸型、发型、体态、服装颜色材质上明显不同
禁止漂移：不要复用同项目其他角色的脸型、发型、服装模板；不要年龄漂移或职业漂移

【镜头功能】
人格锚点：场景功能、临时反应。
对白气质：短句、职业化。
