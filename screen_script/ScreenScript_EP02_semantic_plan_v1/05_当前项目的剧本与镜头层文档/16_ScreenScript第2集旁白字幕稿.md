# ScreenScript第2集旁白字幕稿

> 项目标题：ScreenScript

- SH01：字幕：四年前 夜幕下的半岛酒店灯火辉煌。门口豪车排列如龙，红毯从门口一直铺到大堂。巨型LED屏幕上滚动着——不可读文字块。
- SH02：少来。你才好看，这条裙子起码六位数吧？
- SH03：今晚是陆氏集团的晚宴，你知道陆景琛会来对不对？
- SH04：陆景琛和我们有什么关系。好好干活，别走神。
- SH05：知道了。
- SH06：但赵一鸣注意到了——陆景琛的步子，慢了半拍。 休息时间，沈念歌靠在墙上喝水。手机屏幕亮着——一条未读消息：爸，这个月生活费我已转到你卡上。你在学校好好学习，别打
- SH07：沈念歌看着消息，嘴角弯了弯。她父亲沈远山虽然已经不再做生意了，但每个月都会给她转生活费。她不知道父亲经历了什么，只知道几年前家里不可读文字块，从大房子搬到了小公
- SH08：来，喝杯果汁。今晚你站太久了。
- SH09：谢谢。
- SH10：念歌，我跟你说，陆景琛真的来了。刚才进门的时候我差点被他帅到——
- SH11：你不是有男朋友吗？
- SH12：……那你喜欢他吗？
- SH13：怎么……突然有点晕……
- SH14：一张大床。两个人。被子裹着两个模糊的身影。 沈念歌先醒了。她睁开眼，头痛欲裂。天花板是陌生的——雕花吊灯，丝绸墙纸。这不是她的房间。
- SH15：……谁？
- SH16：我不需要。
- SH17：我说了。我不需要。告诉你们陆少……那晚的事，我什么都不会说。你放心。
- SH18：爸……我没事。我……我想回家。 / 下一集，秘密继续逼近。
