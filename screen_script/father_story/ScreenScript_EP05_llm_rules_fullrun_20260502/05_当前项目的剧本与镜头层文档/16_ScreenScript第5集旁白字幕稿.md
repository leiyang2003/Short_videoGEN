# ScreenScript第5集旁白字幕稿

> 项目标题：星辉幼儿园

- SH01：星辉国际幼儿园——滨海市最顶尖的贵族幼儿园。铁艺大门，修剪整齐的草坪，教学楼的玻璃幕墙在阳光下闪闪发光。门口停满了豪车。
- SH02：沈念歌牵着予予站在幼儿园门口。其他家长穿着名牌、开着豪车，和她格格不入。
- SH03：一个穿着高定风衣的妈妈牵着女儿走过来，瞥了沈念歌一眼。
- SH04：这个人怎么进的星辉？赞助费一年二十万，她穿得像刚从菜市场来的。
- SH05：沈念歌听到了。她低头看了一眼自己的白T恤和帆布鞋——予予的手术费预付款已经掏空了所有积蓄，她连一件新衣服都买不起。
- SH06：但她没有低头。她牵紧予予的手，走进了幼儿园。
- SH07：妈咪，你在想什么？
- SH08：在想予予今天要好好表现。
- SH09：放心吧！我最厉害了！
- SH10：教室宽敞明亮，彩色墙壁，每个孩子有自己的小柜子。十固定数量孩子和他们的家长坐成一圈，老师站在中间。
- SH11：今天是我们中班的第一天，请小朋友们轮流做自我介绍好不好？先说名字，再说一个关于自己的秘密。
- SH12：孩子们一个个轮流。有的说喜欢吃草莓，有的说家里有游泳池，有的说暑假去了巴黎。
- SH13：轮到予予。 他站起来，背带裤的背带松了一边，头发有点翘。但站得很直。
- SH14：大家好，我叫沈知予。小名予予。
- SH15：予予的秘密是什么呀？
- SH16：我爸爸是这个城市最厉害的人。但我还没见过他。等我找到他，他会变得很厉害。
- SH17：教室里安静了一秒。 固定数量家长互相看了一眼，表情微妙。那个风衣妈妈小声笑了一下。
- SH18：沈念歌坐在角落的家长席上，手指掐进了掌心。
- SH19：予予的爸爸一定很了不起。那你有没有什么爱好呢？
- SH20：我喜欢画画。还有数学。100以内的加减法我都能算。
- SH21：真的吗？那老师考考你。37加58等于多少？
- SH22：95。
- SH23：老师愣住了。旁边一个戴眼镜的小男孩掰着手指算了半天还没算出来。
- SH24：这个很简单呀。
- SH25：教室里其他家长的表情变了。从之前的微讽变成了惊讶——甚至有人掏出手机看了一眼这个孩子的资料。，一部手机首帧可见，屏幕朝向持有者，屏幕内容不可见
- SH26：沈念歌看着儿子，嘴角弯了一下，但很快抿回去。她在心里叹了口气——予予太聪明了。聪明的孩子，容易被人记住。被人记住，就容易出事。
- SH27：放学。家长们开着车来接孩子。豪车一辆接一辆驶过。
- SH28：沈念歌站在路边，等公交车。
- SH29：予予背着小书包，从幼儿园里跑出来。
- SH30：妈咪！今天老师说我是全班最聪明的！还给我多发了一颗小星星！
- SH31：真棒。
- SH32：妈咪，我告诉你一个秘密。
- SH33：什么秘密？
- SH34：我今天在幼儿园走廊上看到一张照片。挂在荣誉墙上。一个穿西装的男人，下面写着'陆氏集团总裁陆景琛先生捐赠'。妈咪——就是医院的那个叔叔！
- SH35：沈念歌的表情僵了一瞬。
- SH36：嗯……可能是吧。 / 我就知道！他就住在这个城市！妈咪，你今天不是去他上班的地方了吗？你有没有看到他？ / 走了，公交车来了。
