# ScreenScript第3集旁白字幕稿

> 项目标题：四年前那个夜晚（下）

- SH01：字幕：四年前 · 两个月后 一间不到二十平的小公寓。墙皮有些剥落，窗帘是地摊上最便宜的那种。但收拾得很干净，桌上摆着一小盆绿萝。
- SH02：我不知道你是谁的。但你是我的。
- SH03：妇幼保健院门口。沈念歌的肚子已经很大了，她一个人撑着腰走出门，另一只手拎着一个编织袋。 没有家属签字，没有陪产。产房里只有她一个人。
- SH04：予予……妈妈给你取的名字，知予。知道的知，给予的予。妈妈知道的一切，都想给你。
- SH05：父亲不详。
- SH06：我说了。不详。
- SH07：【画面一】深夜，一间出租屋。沈念歌一手抱着哭闹的予予，一手在电脑上打字做兼职翻译。桌上是泡面和几罐奶粉。时钟指向凌晨三点。 【画面二】清晨，菜市场。沈念歌蹲在菜
- SH08：祝你生日快乐……祝你生日快乐……
- SH09：不能碰，会烫。
- SH10：妈咪，那个房子好大！我们以后能住那么大的房子吗？
- SH11：我爸爸……在很远的地方。但我觉得他会来找我。
- SH12：予予趴在桌上画画。沈念歌坐在旁边看手机——滨海市儿童医院心内科的预约确认信息。，一部手机首帧可见，屏幕朝向持有者，屏幕内容不可见
- SH13：滨海市？就是那个有大高楼的城市？
- SH14：妈咪，滨海市……是不是我爸爸在的地方？
- SH15：我觉得是。我有预感。
- SH16：四年了。我以为这辈子都不会再回到那个城市。可是予予需要最好的医生。而最好的医生……在滨海市。
- SH17：妈咪你看！这是我们一家三口！中间这个黑色的是爸爸，因为我还没见过他！等见到了我就涂成肉色！
- SH18：好。等找到了，你涂成最漂亮的颜色。
