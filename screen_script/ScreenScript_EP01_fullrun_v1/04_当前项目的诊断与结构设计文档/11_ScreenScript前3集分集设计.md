# ScreenScript前3集分集设计

> 项目标题：你是爸爸吗

## 第1集：你是爸爸吗
- 本集目标：陆景琛下令调查沈念歌母子，并安排DNA检测——他为什么要查一个萍水相逢的女人？他和沈念歌之间到底有什么过往？四年前那个夜晚，即将浮出水面。
- 冲突：本集为开篇，无前情
- 情绪点：相遇、萌宝、心动
- 集尾钩子：陆景琛下令调查沈念歌母子，并安排DNA检测——他为什么要查一个萍水相逢的女人？他和沈念歌之间到底有什么过往？四年前那个夜晚，即将浮出水面。

## 第2集：四年前那个夜晚（上）
- 本集目标：沈念歌被设计灌醉送入陆景琛房间，醒来后被管家甩支票赶走——而她不知道的是，陆景琛当晚也被下了药，同样不记得发生了什么。两个被操控的人，以为自己是"自愿的"。真正的幕后黑手，究竟是谁？
- 冲突：陆景琛在医院偶遇沈念歌母子，萌宝一句"你是爸爸吗"震动了他。他下令调查，安排DNA检测。
- 情绪点：闪回、慈善晚宴、陷阱
- 集尾钩子：沈念歌被设计灌醉送入陆景琛房间，醒来后被管家甩支票赶走——而她不知道的是，陆景琛当晚也被下了药，同样不记得发生了什么。两个被操控的人，以为自己是"自愿的"。真正的幕后黑手，究竟是谁？

## 第3集：四年前那个夜晚（下）
- 本集目标：沈念歌独自生子、四年辗转打工养娃的艰辛已经让人心疼，而她带着儿子回到滨海市的原因只有一个——予予的心脏手术。她不知道的是，那张她扔掉的五十万支票，在四年后将成为两个人命运再次交汇的伏笔。
- 冲突：四年前慈善晚宴上，沈念歌被闺蜜林雨薇设计灌醉，送入陆景琛房间。次日醒来，管家甩来五十万支票让她离开。她一分未取，赤脚走出了那扇门。
- 情绪点：怀孕、独自生子、归来
- 集尾钩子：沈念歌独自生子、四年辗转打工养娃的艰辛已经让人心疼，而她带着儿子回到滨海市的原因只有一个——予予的心脏手术。她不知道的是，那张她扔掉的五十万支票，在四年后将成为两个人命运再次交汇的伏笔。
