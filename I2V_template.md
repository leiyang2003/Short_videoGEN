# I2V Prompt Template

```text
{时间}，{地点}，低饱和写实电影感，9:16竖屏。

首帧：{首帧画面描述}
{必须从第一帧可见的人物、脸部可见性、关键道具、道具数量、道具位置、环境状态。}
{明确禁止新增道具、字幕、文字、水印。}

动作：{镜头运动与人物动作}
{说明镜头从哪里开始、如何运动、最后停在哪里。}
{说明每个可见人物是否移动、是否说话、是否触碰道具。}
{说明关键道具全程固定不移动、不新增、不消失、不漂移。}

旁白：{旁白内容；如果没有旁白，写“无旁白”。}

画面内声音：{画面内人物声音、环境声、道具声。}
{如果不是完整台词，明确写“不说完整台词”。}

镜头：{景别}，{镜头焦段}，{视角}，{摄影机运动}。
{最后的构图焦点与拉近/停留位置。}
写实电影光影，稳定连续，无水印，无文字。
```

## Example

```text
清晨，东京银座高级酒店套房，低饱和写实电影感，9:16竖屏。

首帧：服务员站在门口，背影在画面左前景，房间深处，佐藤彩花静止躺卧在凌乱的白色床铺上，脸部可见，浅粉色丝巾固定在颈部。落地窗外是清晨城市 skyline，晨光洒入房间。床头柜和圆桌上可见威士忌瓶与一只玻璃杯，房间凌乱但没有新增道具。

动作：镜头从画面左侧门口开始，缓慢向右横移，越过服务员的肩线和半开的房门，逐渐把视觉重心从门口转移到床上。镜头最后停在佐藤彩花身上，拉进镜头聚焦到她的脸部与颈部丝巾。服务员只保持震惊凝视，不走入房间；彩花全程静止，丝巾、酒瓶、玻璃杯全程固定不移动。只表现发现现场的惊惧氛围，不出现字幕。

旁白：清晨，服务员推开房门，空气里还残留着茉莉香水与威士忌的余味。她的目光越过凌乱的床铺，停在佐藤彩花身上。

画面内声音：服务员屏住呼吸，发出一声惊叫，不说完整台词。

镜头：大全景建立镜头，24mm，平视角度，缓慢 left-to-right dolly / lateral pan，最后拉进镜头到佐藤彩花脸部和浅粉色丝巾。写实电影光影，稳定连续，无水印，无文字。
```
