## 4. 详细对比

### SH05（送粥）
- 改善点（B优于A）：
  - 男主在前景的清晰露脸显著增加，达到“可辨识窗口”目标。
- 回退点（B劣于A）：
  - 男主发型/气质与 SH02 锚点不一致（A中虽虚焦，但基调更接近“破败、消瘦、狼狈”）。
  - 女主服装整洁度与“底层困境”设定略有漂移。

评分（10分制）：
- 露脸可辨识度：A `6.8` -> B `9.1`（+2.3）
- 男主跨镜头锚点一致性：A `8.0` -> B `6.2`（-1.8）
- 镜头整体可用性：A `8.7` -> B `7.8`（-0.9）

判定：**部分成功但不可直接升级为新基线**。

### SH10（盐渍）
- 改善点（B优于A）：
  - A中的“雪花/放射状图案”被有效抑制。
  - B中的盐渍呈不规则颗粒与晶块，符合自然沉积逻辑。
  - 质感更贴近“写实发现机会点”。
- 未见明显负面回退。

评分（10分制）：
- 盐渍写实可信度：A `6.4` -> B `8.9`（+2.5）
- 关键道具识别清晰度：A `8.6` -> B `8.8`（+0.2）
- 镜头整体可用性：A `8.3` -> B `9.0`（+0.7）

判定：**优化成功，建议B替换A**。
