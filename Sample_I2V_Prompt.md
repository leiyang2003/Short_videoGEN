# Sample I2V Prompt

This is the standard prompt shape for Seedance image-to-video shots after the prompt renderer update.

Rules:

1. Inherit scene anchors from the keyframe `start/prompt.txt` whenever available.
2. Use a structured prompt instead of a freeform one-line prompt.
3. For dialogue shots, always render:
   - `场景`
   - `角色锁定`
   - `画面主体`
   - `镜头与表演执行`
   - `台词与嘴型必须严格对应`
   - `禁止`
4. Non-speaking characters must explicitly stay silent and keep their mouths closed.

Example for `SH12`:

```text
[场景：古代中国西汉背景，竖屏短剧，写实电影感，低饱和，冷硬底色，泥土感，真实光影，角色一致性稳定，握陶罐立决心，夜色溪边、远景深色背景，半身近景，冷色主导，轮廓光]

角色锁定：
- 林辰：18-22岁，乱世底层逆袭俊美少年，消瘦劲健，宽肩窄腰，冷峻英气，灰褐旧麻布衣，低饱和写实；精致长椭圆脸；狭长锐利丹凤眼；直挺鼻梁；薄唇；轮廓立体俊朗；中长黑发凌乱束于脑后，碎发垂落；黑色；白皙中带风霜浅麦色与尘土颗粒感；灰褐粗麻短褐，肩肘处有旧补丁，边缘毛糙；暗米麻布中衣，领口磨损发旧；低沉沙哑男声；不认命、主导。
- 阿翠：16-18岁，清冷绝美少女，纤细匀称，隐忍高洁，灰青旧麻布衣，低饱和写实；小巧精致椭圆脸；清澈杏仁眼；小巧直鼻；薄樱唇；眉目柔美；低位黑发挽简洁发髻，仅布绳束发；黑色；白皙细腻，受风寒略带自然粗糙感；灰青粗麻长衫，袖口下摆轻磨损；浅灰麻布里衣，领口素净；轻柔克制女声；跟随。

画面主体：消瘦少年手握破陶罐抬头看向远处，虚弱中带攻击性，夜色更深，翻身决心显化

镜头与表演执行：
- 运动：缓慢拉近
- 构图焦点：陶罐与目光
- 动作意图：林辰握陶罐抬头，发出行动指令
- 情绪意图：翻盘决心
- 关键道具：破陶罐
- 连续性：道具：陶罐持续可见；握陶罐立决心连续；关键道具保持：破陶罐

手部约束：
- 人物手部解剖必须真实稳定，每只可见的手都清晰呈现五根手指。
- 禁止缺指、四指、多指、并指、手指粘连、手部扭曲、手掌变形、手部畸形。
- 这是手部重点镜头，主手必须完整、清晰、稳定入镜，五根手指清楚可辨。
- 指节、指腹、指甲、掌缘结构自然，手指之间自然分开，不能融合或缺失。
- 不要用遮挡、裁切、运动模糊掩盖手指数量错误。

台词与嘴型必须严格对应：
- 0.5-2.9秒：林辰开口说：“阿翠，明早跟我走。” 阿翠不说话，不张口，只保持沉默反应。
- 2.9-3.3秒：短暂停顿，情绪延续。
- 3.3-4.5秒：阿翠开口说：“去哪儿？” 林辰不说话，不张口，只保持沉默反应。
- 4.5-4.9秒：短暂停顿，情绪延续。
- 4.9-5.5秒：林辰开口说：“挣活路。” 阿翠不说话，不张口，只保持沉默反应。

禁止：
- 不要让阿翠说林辰的台词。
- 不要让林辰说阿翠的台词。
- 不要新增台词。
- 不要省略台词。
- 非说话人保持闭嘴。
- 不要出现现代服装、现代建筑、现代道具。
```

- <角色B>：...

镜头与表演执行：
- 运动：<movement>
- 构图焦点：<framing_focus>
- 动作意图：<action_intent>
- 情绪意图：<emotion_intent>
- 关键道具：<props>
- 连续性：<continuity_rules>

手部约束：
- 人物手部解剖必须真实稳定，每只可见的手都清晰呈现五根手指。
- 禁止缺指、四指、多指、并指、手指粘连、手部扭曲、手掌变形、手部畸形。
- <若手部是视觉重点，则追加：手部特写要求与五指清晰可辨要求>

台词与嘴型必须严格对应：
- <start>-<end>秒：<speaker>开口说："..."
- <pause_start>-<pause_end>秒：短暂停顿，情绪延续。

禁止：
- 不要让<角色B>说<角色A>的台词。
- 不要新增台词。
- 不要省略台词。
