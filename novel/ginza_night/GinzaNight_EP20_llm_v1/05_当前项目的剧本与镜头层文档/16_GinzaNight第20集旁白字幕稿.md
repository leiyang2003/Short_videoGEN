# GinzaNight第20集旁白字幕稿

> 项目标题：银座的夜晚与隐藏的温柔（1-6章）

- SH01：建立第20集开场的午后警视厅办公室压力，把石川与案件核心证据放在同一张办公桌前。
- SH02：用四份排除嫌疑材料明确龙崎、山田、阿彻、太郎四条线索已被证据排除，把证据压力继续收束。
- SH03：用石川停在卷宗页上的凝视，把证据链指向美咲与逮捕可能伤害樱子的两难外化。
- SH04：健一先生，最后确认。自白细节呢？
- SH05：那夜我在房间。丝巾是我缠的。彩花靠着我，说起小樱，我失控了。
- SH06：为什么？
- SH07：DNA的事，我知道。小樱是我的血脉。我信您，刑警先生，让她走。
- SH08：（脚步声渐远，金属门轻响）
- SH09：姨妈，我们真要走吗？妈妈的照片怎么办？
- SH10：樱子，妈妈爱你，可那爱藏着痛。 / 小时候，姐姐拉我进牺牲的循环；你才十四岁，我不会让它重来。 / 我们走，重新开始。
- SH11：姨妈，妈妈为什么……要献身？
- SH12：因为爱。但错了。我们断它。从今开始。
- SH13：用按灭香烟与合上卷宗的静默动作，表现石川明知证据链完整，却选择沉默，默许美咲带樱子离开。
