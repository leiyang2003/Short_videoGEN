# GinzaNight第4集旁白字幕稿

> 项目标题：GinzaNight

- SH01：石川悠一：山田先生，手别一直敲。我们只是确认几件事。
- SH02：石川悠一：彩花的手机里有一部匿名小说，叫《银座的夜》。里面写到您。 / 山田老先生：写到我？那是谁胡写的？
- SH03：佐藤美咲：这是姐姐的手机，我一直拿着。
- SH04：石川悠一：田中先生，彩花提过这部小说吗？ / 田中健一：没有。她只说工作累，偶尔写点东西解闷。
- SH05：田中健一：这些句子……她没有告诉过我。
- SH06：石川悠一：男人解开领带，女人低语承诺。山田先生，这像您吗？ / 山田老先生：荒唐！那只是小说，虚构的！
- SH07：山田老先生：我现在就打给我妻子。 / 山田妻子（电话）：是的，当晚他在医院陪我，护士可以证明。 / 石川悠一：记录下来。
- SH08：佐藤美咲：姐姐偶尔写日记，我不知道那被叫成小说。也许只是发泄。
- SH09：田中健一：小樱也在这些文字里吗？ / 佐藤美咲：田中先生，别把小樱扯进来。
- SH10：山田老先生：我去过酒店几次，只是聊聊。我没理由害她。 / 石川悠一：今天先到这里。
- SH11：石川悠一：未完章写到家族，也写到小樱。 / 佐藤美咲：那不是给你们看的。
- SH12：石川悠一：暂时排除山田。接下来查阿彻的行车记录。
- SH13：佐藤美咲：田中先生，到这里就够了。 / 田中健一：不够。那份匿名线索，已经写到小樱的真实处境了，对吗？
