# GinzaNight第3集旁白字幕稿

> 项目标题：GinzaNight

- SH01：石川悠一：龙崎先生，昨晚十点，你在哪里？
- SH02：石川悠一：这段监控里，你从黑车下来，走向酒店侧门。
- SH03：龙崎：巧合而已。我去见客户，不是那家酒店。照片和证人都在这里。
- SH04：石川悠一：田中先生，来警视厅一趟。需要你提供些细节。 / 田中健一：我现在过去。
- SH05：石川悠一：田中先生，你和佐藤彩花的关系如何？ / 田中健一：常客而已。她人很好，总听我抱怨工作。
- SH06：佐藤彩花：龙崎想拉我过去，说条件更好。但我不会走。 / 田中健一：为什么告诉我这些？
- SH07：石川悠一：你姐姐最近提过龙崎吗？ / 佐藤美咲：彩花姐工作忙，最近压力大。龙崎的事，她提过几次，说是生意纠纷。
- SH08：龙崎：我没理由害她！拉她过去是生意，她拒绝了而已。 / 另一家店经理：龙崎先生昨晚在我们店，时间对得上，店员也能证明。
- SH09：石川悠一：暂时排除你。把监控全拷贝，一帧别漏。
- SH10：龙崎：我真的只是做生意。
- SH11：田中健一：佐藤小姐，你是不是知道什么？ / 佐藤美咲：我没什么能说的。
- SH12：田中健一：那句守护，到底是信任，还是把我推进网里？
- SH13：石川悠一：去查山田老先生。
