# GinzaNight第22集旁白字幕稿

> 项目标题：银座的夜晚与隐藏的温柔（1-6章）

- SH01：以清晨东京站站台建立终章余波，呈现石川在列车远去后的冷静转身，并导向警视厅最终判断线。
- SH02：把石川从清晨离站后的外部行动转入警视厅办公室的最终判断线，首次呈现案件卷宗并让他克制地打开。
- SH03：用桌面证据展示三张案件照片与四页嫌疑人排除资料，确认命案现场线索和四层嫌疑排除链条。
- SH04：在四名嫌疑人已被排除后，用石川面对邻室记录页的无声停顿，把真相重心转向美咲与彩花的家庭链条。
- SH05：就这样吧。
- SH06：用一个克制的封存动作落实石川悠一停止继续追查美咲的选择，让沉默替代判决。
- SH07：在留置所牢房中用无声反应回收健一对领带记忆与樱子身世的承担，把上一场石川封存真相后的代价落到健一身上。
- SH08：彩花，对不起。但小樱，会新生。
- SH09：姨妈，那边……银座，还在吗？
- SH10：小樱，不会了。 / 妈妈的爱，是真的，但藏着痛。 / 我们走，远离开它。新生，从这里开始。
- SH11：姨妈，健一叔叔……他会好吗？
- SH12：会。他托我带你走，守护你。 / 姐姐曾把我拉进那世界，说献身要付出一切，可那是错的。 / 你才十四岁，姨妈不会让它再发生。
- SH13：银座恢复平静，真正的夜晚留在每个人心里。
