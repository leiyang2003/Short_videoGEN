# GinzaNight第8集旁白字幕稿

> 项目标题：GinzaNight

- SH01：田中健一：彩花，你到底留下了什么给我？
- SH02：田中健一：怎么会有照片？
- SH03：佐藤彩花：如果我没了，小樱就拜托你。 / 田中健一：小樱……原来是你。
- SH04：田中健一：你从没告诉过我，你有女儿。
- SH05：田中健一：我是她的守护人，还是你早就摆好的棋子？
- SH06：田中健一：谁在外面？
- SH07：佐藤樱子：田中叔叔，我……我想来看看妈妈的东西。
- SH08：田中健一：进来吧。坐吧，慢慢说。 / 佐藤樱子：谢谢，田中叔叔。
- SH09：佐藤樱子：这是妈妈的。我偷看过她用，晚上回家，她总在镜子前涂唇膏。
- SH10：田中健一：她很爱你。那些，是她的工作。 / 佐藤樱子：工作？银座是什么样的？她总晚回家，我好奇，又怕。
- SH11：佐藤美咲：小樱，别碰那些。妈妈的东西，会让人乱想。 / 佐藤美咲：田中さん，谢谢你收留这些。但别深挖，小樱还小。
- SH12：佐藤樱子：叔叔，你见过妈妈在银座的样子吗？她开心吗？ / 田中健一：她辛苦。但有你，她开心。 / 佐藤美咲：够了，小樱，我们走吧。彩花的秘密别告诉她，家族的事很复杂。
- SH13：石川悠一：田中先生，指纹比对结果已出。来一趟吧。 / 田中健一：指纹……所以，我只是她安排好的工具吗？
