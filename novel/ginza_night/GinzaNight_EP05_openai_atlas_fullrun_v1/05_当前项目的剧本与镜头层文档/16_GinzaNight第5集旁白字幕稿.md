# GinzaNight第5集旁白字幕稿

> 项目标题：GinzaNight

- SH01：石川悠一：阿彻先生，昨晚的记录我们看了。酒店附近，你转了三圈。
- SH02：石川悠一：看清楚，这是同一家酒店的街角。
- SH03：警员：田中先生，请坐那边。石川刑警要你确认跟踪的事。
- SH04：石川悠一：你见过这个人吗？彩花提过陌生人靠近？ / 田中健一：提过一次。她说有人在酒店外等，眼神怪怪的，让我小心。
- SH05：石川悠一：阿彻先生，解释吧。为什么徘徊？ / 阿彻：我喜欢她。但我没下车，没靠近。
- SH06：阿彻：十点到十二点，我在停车场，没动过。
- SH07：警员：请确认阿彻先生当晚的时间。 / 酒吧朋友：十点到十二点都没离开。
- SH08：阿彻：我没害她。她人好，从不嫌我车旧。
- SH09：石川悠一：这是小樱，彩花的女儿。你见过她吗？
- SH10：田中健一：没有。但彩花说过，要我帮她守护什么。或许就是她。 / 佐藤彩花：如果我没了，小樱……
- SH11：佐藤美咲：小樱没事吧？ / 田中健一：没事。只是照片。她看起来……像彩花。 / 佐藤美咲：别让小樱靠近那些东西。银座的影子，太长了。
- SH12：田中健一：彩花的世界，不能再把小樱卷进去。
- SH13：石川悠一：暂时排除阿彻。看侧门记录。 / 石川悠一：停，放大邻室门口。这个侧影……像佐藤美咲。
