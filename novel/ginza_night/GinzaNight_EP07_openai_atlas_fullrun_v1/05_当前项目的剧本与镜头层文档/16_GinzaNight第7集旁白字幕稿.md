# GinzaNight第7集旁白字幕稿

> 项目标题：GinzaNight

- SH01：鉴定员：纹路和颈部红印完全吻合。 / 石川悠一：确认了。凶器就是这条丝巾。赠送者，太郎。
- SH02：警员：田中先生，请进。石川刑警要你确认细节。 / 田中健一：我知道。
- SH03：石川悠一：彩花怎么用这条丝巾？ / 田中健一：她会缠在颈间……手指，还会碰我的领带。
- SH04：佐藤彩花：这丝巾，舒服。
- SH05：石川悠一：太郎先生，这丝巾，你送的吧。去年冬天，彩花生日。
- SH06：太郎：是。我是想恩返。她帮我卖了第一幅画，我才赚到第一笔钱。我挑浅蓝的，像她的眼睛。
- SH07：石川悠一：动机呢？她拒绝了你的追求？ / 太郎：没有。我没去酒店。那晚我在直播，朋友都在。
- SH08：警员：时间戳从十点到凌晨，画面连续。 / 太郎的朋友：对，太郎一直在那儿画画，眼睛红红的，还提到彩花小姐。
- SH09：太郎：我没害她。她是恩人。她收到丝巾时，笑着缠上脖子，对我说谢谢。
- SH10：石川悠一：记录到这里。太郎先生，先出去。 / 太郎：我没做。真的。
- SH11：佐藤樱子：姨妈，妈妈的丝巾，我见过。她缠着时，总笑得奇怪。 / 佐藤美咲：别碰。远离那些。
- SH12：石川悠一：太郎排除。你的回忆，让丝巾痕迹对上了。 / 田中健一：下一个焦点，是什么？ / 石川悠一：是你。指纹比对，结果快出。
- SH13：田中健一：抽屉里……也许还有答案。 / 佐藤美咲：丝巾上的细节指向熟悉彩花的人。
