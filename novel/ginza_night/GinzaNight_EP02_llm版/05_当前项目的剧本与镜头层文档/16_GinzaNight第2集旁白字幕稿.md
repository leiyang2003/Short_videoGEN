# GinzaNight第2集旁白字幕稿

> 项目标题：GinzaNight

- SH01：午后的阳光刺得他睁不开眼。 / 那条领带，把昨夜又拽了回来。
- SH02：同一间套房里，茉莉香水混着陈年威士忌。 / 烛光把他们的影子拉得很长。
- SH03：佐藤彩花：健一，你总是这么可靠。
- SH04：佐藤彩花：如果我消失了，你会守护小樱吗？ / 田中健一：当然，我会。
- SH05：田中健一：小樱……守护？ / 苦涩在舌尖散开，那些词却更清楚了。
- SH06：酒店服务员（电话）：佐藤小姐，彩花小姐出事了。 / 她的第一反应不是哭，是手指突然失去力气。
- SH07：姐姐的温柔像一张网。 / 美咲知道，小樱不能再被拖进去。
- SH08：石川悠一（门内）：勒痕不像自缢，丝巾要再比对。 / 警员（门内）：是，石川先生。
- SH09：佐藤彩花（回忆）：守护小樱。 / 田中健一：小樱……是你吗？ / 敲门声突然响起。
- SH10：石川悠一：龙崎先生，昨晚十点左右，你在酒店附近出现过。
- SH11：龙崎：巧合而已。我去见客户，不是那家酒店。 / 警员：另一家店经理确认，时间对得上。 / 石川悠一：暂时排除你。把监控全拷贝。
- SH12：石川悠一：田中先生，你和佐藤彩花的关系如何？ / 田中健一：常客而已。她人很好，总听我抱怨工作。 / 佐藤彩花（回忆）：龙崎想拉我过去，说条件更好。 / 佐藤美咲：彩花姐最近压力大。龙崎的事，她说是生意纠纷。
- SH13：警员：石川先生，田中健一的离开时间，对不上。 / 石川悠一：他说没待太久，可记录不是这样。 / 警方发现健一的说法与酒店记录不一致。
