# GinzaNight第21集旁白字幕稿

> 项目标题：银座的夜晚与隐藏的温柔（1-6章）

- SH01：以清晨东京站新干线月台的薄雾与冷银铁轨建立本集离开银座前的出发空间。
- SH02：让美咲和樱子首次进入本集画面，以牵手慢行外化守护关系，并把观众视线从清晨月台引向新干线检票口。
- SH03：姨妈，那边……妈妈的世界，还在等吗？
- SH04：小樱，不会重复那些错误。 / 妈妈的爱是真的，但藏着痛。我们走，去新生，从这里断开。
- SH05：承接美咲上一镜的坚定回应，只用无声反应呈现樱子收回牵挂、靠近美咲并开始信任她。
- SH06：守护孩子，比抓犯人更重要。
- SH07：姨妈，爸爸的信……他说会好吗？
- SH08：健一叔叔会的。他托我带你走，远离开链条。小樱，你是希望，别怕。
- SH09：（站内铃声响起） / （列车低鸣从雾中传来）
- SH10：以石川悠一的沉默离场完成默许守护的收束，承接美咲和樱子过闸后的远距离旁观。
- SH11：完成美咲和樱子从月台跨入新干线车厢的动作节点，视觉上从旧世界进入新空间。
- SH12：从今开始，一切新生。
- SH13：完成第21集集尾视觉钩子：列车已启动，车窗外月台与银座方向的灰蓝都市轮廓在清晨薄雾中后退、淡化，强调两人离开银座旧世界。
