# GinzaNight第2集旁白字幕稿

> 项目标题：GinzaNight

- SH01：田中健一：这条领带……昨晚她也碰过。
- SH02：佐藤彩花：今晚这里只有我们两个，别那么紧张。
- SH03：佐藤彩花：健一，你总是这么可靠。
- SH04：佐藤彩花：如果我消失了，你会守护小樱吗？ / 田中健一：当然，我会。
- SH05：田中健一：小樱……守护……你到底想让我做什么？
- SH06：酒店服务员：佐藤小姐，彩花小姐出事了，警方已经到酒店了。 / 佐藤美咲：你说什么？姐姐她怎么了？
- SH07：佐藤美咲：小樱的事，绝不能被他们问出来。
- SH08：石川悠一：勒痕不是普通自杀留下的，丝巾的位置也不自然。 / 佐藤美咲：丝巾……他们已经注意到了。
- SH09：田中健一：这是你前几天留下的……不是巧合，对吗？
- SH10：田中健一：十四岁……小樱就是她？谁在外面？
- SH11：石川悠一：龙崎先生，昨晚十点，你在酒店附近出现过。 / 龙崎：巧合而已。我去见客户，不是那家酒店。这里有合影，时间也对得上。 / 警员：经理电话确认，他那晚确实在另一家店。
- SH12：石川悠一：田中先生，你和佐藤彩花的关系如何？ / 田中健一：常客而已。她人很好，总听我抱怨工作。 / 佐藤美咲：彩花姐工作忙，最近压力大。龙崎的事，她提过几次，说是生意纠纷。
- SH13：警员：石川刑警，酒店出入记录核完了。田中健一前夜二十三点十七分，单独进过彩花的套房。 / 石川悠一：他说只是常客。记录和说法，不一致。
